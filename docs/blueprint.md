# **App Name**: Neighbor Nexus

## Core Features:

- User Verification: Verify users' addresses to ensure they are within the designated neighborhood radius.
- Tool Listing and Search: Allow users to list tools they own and make them available for rent or loan, including search and filtering capabilities.
- Skill Listing and Search: Allow users to list their skills and make them available for exchange or hire, including search and filtering capabilities.
- Transaction Management: Enable users to request, approve, and track tool rentals and skill exchanges.
- Review and Rating System: Allow users to rate and review their experiences with tool rentals and skill exchanges to encourage responsible behavior.

## Style Guidelines:

- Primary color: Deep turquoise (#45B8AC) to evoke trust and neighborliness. Use Material UI's theme provider to set this color as primary.
- Background color: Light cyan (#E0F4F1) to create a clean and open feel. Use Material UI's theme provider to set this color as background.
- Accent color: Soft coral (#F08A5D) for call-to-action buttons and highlights. Use Material UI's theme provider to set this color as secondary.
- Body and headline font: 'PT Sans', a humanist sans-serif, to offer a modern look and a little warmth. Use Material UI's typography variants and customize the font family.
- Use clear and simple icons representing tools, skills, and community engagement, such as gears, hands shaking, and houses. Utilize Material UI's Icon component and its library of icons.
- Implement a card-based layout for listings and profiles to create a visually appealing and user-friendly browsing experience. Use Material UI's Card component for this purpose.
- Use subtle animations, like fade-ins and transitions, to enhance user engagement without overwhelming the interface. Use Material UI's Transition component or other animation libraries compatible with React and Material UI.
